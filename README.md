Since my last push, I made several updates to the app. The login form is now functional and connected to the database. The profile page is still a work in progress. Some features are static for now, but others are working dynamically, such as the reviews section (there is still a lot to add, though the API is working when tested in Postman). The user's name is also fetched from the database.
